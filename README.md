## 목적

*세상에 소외 받는 곳이 있습니다. 이 앱은 소외 받는 곳의 데이터를 모아줍니다.* 

*소외 받는 곳의 첫번째는 장애인 화장실입니다.*

*장애인 화장실의 상세 데이터를 수집하여 장애인 화장실의 사용율을 높일 것입니다.*
